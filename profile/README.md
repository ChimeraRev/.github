# Chimera
Chimera is a project to resurrect old builds of Roblox. You've probably heard of this "Old Roblox Revival" concept before.

We have made some advanced tools to easily get these old builds running with our own private servers.

With our currently existing tools, theoretically any version of Roblox could be patched, but however we are currently focusing on 2015 builds of Roblox.

Currently the project is very early in development, and is not fully ready for a public release.
